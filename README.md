# .emacs.d

## install

```sh
$ mkdir ~/emacs
$ git clone git@github.com:bcfurtado/.emacs.d.git ~/emacs/emacs-bcfurtado
$ ln -s ~/emacs/emacs-bcfurtado ~/.emacs.d
```

## requirements

- emacs >= 25


## lsp requirements
```
npm install -g javascript-typescript-langserver
npm install -g vue-language-server
```
